# Keyboard Shortcuts

Quick reference for all Markit keyboard shortcuts.

---

## File Operations

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + N` | New File |
| `Cmd/Ctrl + O` | Open File |
| `Cmd/Ctrl + D` | Open Folder |
| `Cmd/Ctrl + S` | Save File |
| `Shift + Cmd/Ctrl + S` | Save As |
| `Cmd/Ctrl + A` | Select All |

## View & Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + /` | Toggle Edit/Preview Mode |
| `Cmd/Ctrl + B` | Toggle File Explorer |
| `Cmd/Ctrl + Tab` | Switch Recent Files |
| `Cmd/Ctrl + ,` | Open Settings |

## Search

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + F` | Local Search (current file) |
| `Cmd/Ctrl + Shift + F` | Global Search (all files) |
| `F3` or `Cmd/Ctrl + G` | Find Next Match |
| `Shift + F3` or `Cmd/Ctrl + Shift + G` | Find Previous Match |

## Window

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + 0` | Reset Zoom |
| `Cmd/Ctrl + +` | Zoom In |
| `Cmd/Ctrl + -` | Zoom Out |
| `Cmd/Ctrl + W` | Close Window (macOS) |

## Editor

| Feature | Description |
|---------|-------------|
| Smart Paste | HTML automatically converts to Markdown |
| Auto-save | Configurable interval in Settings |
| Context Menu | Right-click for file operations |
| Scroll Sync | Editor and preview scroll together |

---

## Tips

- **Recent Files**: Hold `Cmd/Ctrl` and press `Tab` repeatedly to cycle through recent files
- **Search Navigation**: Use `F3` / `Shift + F3` to navigate search results quickly
- **Mode Toggle**: Press `Cmd/Ctrl + /` to quickly switch between editing and preview
